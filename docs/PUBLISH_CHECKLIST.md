# 公開前チェックリスト

公開前に次の項目を確認してください。

## データ

- [ ] `data/survey_data.csv` の件数が意図した件数になっている。
- [ ] 作品名や数値を推測で変更していない。
- [ ] 欠損値を推測で補完していない。
- [ ] `python tools/validate_data.py data/survey_data.csv` がエラーなしで完了する。

## サイト表示

- [ ] `index.html` をブラウザで開ける。
- [ ] ヘッダーは左にイラスト、右に文字の構成になっている。
- [ ] 白いカード枠がヘッダー文字の周りに戻っていない。
- [ ] 検索が動く。
- [ ] 年別・クール別が動く。
- [ ] 作品から探すが動く。
- [ ] ランキングが動く。
- [ ] スマホ幅で表示が崩れていない。

## 公開運用

- [ ] `privacy.html` を確認した。
- [ ] `affiliate-disclosure.html` を確認した。
- [ ] `admin.html` に `noindex` が入っている。
- [ ] 外部CSVのURLをコードに直書きしていない。
- [ ] 外部CSVや一覧データを利用する場合、利用条件と出典表示を確認した。
- [ ] GitHub Actionsを使う場合、`UPSTREAM_CSV_URL` を GitHub Secrets に設定した。
- [ ] 公開先に不要な一時ファイル、キャッシュ、作業フォルダを含めていない。
